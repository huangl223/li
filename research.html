Other research artifacts

  • AutoProof: a program verifier for Eiffel
    I extended the features of AutoProof with counterexample extraction and test generation

  • Proof2Test: a tool that generates and simplifies counterexamples of failed proofs (2023)

    ProTL reads a Boogie file and an SMT-LIB counterexample file and output simple variable and value pairs for the Boogie programs

  • A website that showcases a list of formally verified and deployed systems (2023)

  • ProTL: probablistic CCSL translator (2019)

    A translator that converts CCSL (clock constraint specification language) specifications into stochastic timed automata verifiable in Uppaal SMC

  • ProCCSL Simulink library (2017)

    A set of Simulink patterns that represent the executional semantics of CCSL temporal operators
